Product Content Generator with ChatGPT allows you to connect your WooCommerce store to ChatGPT.

== Description ==

Product Content Generator with ChatGPT is a module, intended to connect Store Manager software with an OpenAI account to generate product descriptions in your online store.
Desktop application [Store Manager for WooCommerce](https://emagicone.com/products/store-manager-for-woocommerce) is a comprehensive solution with ever-increasing set of features that overcome default back-end functionality. Focused on flawless user experience Store Manager is tailored to specific business requirements.

== Minimum Requirements ==

* WordPress version 4.1.1 or higher
* WooCommerce version 2.3.4 or higher
* PHP version 5.4 or higher
* MySQL version 5.0 or higher

Please note, ChatGPT Product Content Generator is not a stand-alone product. It is used to set connection between Store Manager for WooCommerce and your OpenAi account.
If you still do not have Store Manager installed, download free trial [by this link](https://emagicone.com/products/store-manager-for-woocommerce)

== Installation ==

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of WooCommerce Store Manager Connector, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “ChatGPT Product Content Generator” and click Search Plugins. Once you’ve found our plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.


== Manual installation ==

The manual installation method involves downloading our plugin from our official site with further uploading it to your web server via WordPress backend or any FTP client.

== Updating ==

After new version of ChatGPT Product Content Generator rolls out, you can easily update the plug-in from backend. Version number and update button will be immediately displayed in the list of plug-ins.
